## 🚀 Quick Start & Deployment

### ☁️ Option 1: Streamlit Cloud (Free & Recommended)
Deploy your own instance globally in less than 2 minutes.

1. **Fork** this repository.
2. Visit [share.streamlit.io](https://share.streamlit.io/) and connect your GitHub account.
3. Select your fork and set the Main file path to `app.py`.
4. Click **Advanced Settings** and add your API key:
   ```toml
   GROQ_API_KEY = "gsk_your_api_key_here"
   ```
5. Click **Deploy**.

### 💻 Option 2: Run Locally (Development)

```bash
# 1. Clone the repository
git clone https://github.com/Shweta-Mishra-ai/excel-auto-analyst.git
cd excel-auto-analyst

# 2. Install dependencies
pip install -r requirements.txt

# 3. Configure your API key
mkdir -p .streamlit
echo 'GROQ_API_KEY = "gsk_your_key_here"' > .streamlit/secrets.toml

# 4. Boot the platform
streamlit run app.py
```
*Open **http://localhost:8501** in your browser.*

> **Need a Groq API Key?**  
> Get one for free at [console.groq.com](https://console.groq.com)

The application adheres to a strict MVC-style separation of concerns to ensure maintainability and security.

```text
excel-auto-analyst/
├── app.py                      # Main Streamlit router (zero business logic)
├── config/
│   └── settings.py             # Centralized application constants
├── core/
│   ├── data_loader.py          # robust CSV/XLSX stream loading
│   ├── validator.py            # schema detection and profiling
│   └── cleaner.py              # imputation and duplication removal
├── analytics/
│   └── stats_engine.py         # statistical math, outliers, and correlations
├── ai/
│   ├── safe_executor.py        # AST-sandboxed python execution environment
│   └── prompt_builder.py       # LLM prompt injection and schema formatting
├── reports/
│   └── ppt_generator.py        # automated python-pptx generation engine
├── ui/pages/                   # modular UI rendering scripts
└── tests/                      # 60+ unit tests ensuring stability
```

## 🧪 Developer Workflow

We enforce strict formatting and maintain 60+ unit tests to ensure high-quality code.
